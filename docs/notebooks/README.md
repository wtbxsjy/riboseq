# Notebooks

The main teaching notebook is:

- `riboseq_guided_tutorial.ipynb`

It imports:

- `riboseq_guided_tutorial_helpers.py`

Suggested notebook dependencies:

- Python 3
- `pandas`
- `ipywidgets`
- `matplotlib`

Notebook runtime used by this project:

- Python environment: `/home/renzhe/miniforge3/bin/python`
- Jupyter kernel: `Python (riboseq-notebook)`

Open the notebook from within the repository so the helper can locate the repo root and tutorial demo data.

VS Code troubleshooting:

- If you are using WSL / remote Linux, open the repository in the same remote environment.
- Reload the VS Code window after creating or updating the kernel.
- In the notebook kernel picker, prefer `Python (riboseq-notebook)`.
- If VS Code still shows an old kernel such as `nf`, reopen the notebook and re-select the kernel after the notebook metadata has been updated.

# nf-core/riboseq: Documentation

The nf-core/riboseq documentation is split into the following pages:

- [Lab Tutorial](lab_tutorial.md)
  - A beginner-friendly lab-facing vignette covering public data retrieval, samplesheet generation, running the pipeline, and reading outputs.
- [Usage](usage.md)
  - An overview of how the pipeline works, how to run it and a description of all of the different command-line flags.
- [Output](output.md)
  - An overview of the different results produced by the pipeline and how to interpret them.

Interactive teaching notebook:

- [Guided Tutorial Notebook](notebooks/riboseq_guided_tutorial.ipynb)
  - An accession-aware notebook that helps query ENA/NCBI metadata, prepare download commands, build a samplesheet, and assemble a `nextflow run` command.

You can find a lot more documentation about installing, configuring and running nf-core pipelines on the website: [https://nf-co.re](https://nf-co.re)
